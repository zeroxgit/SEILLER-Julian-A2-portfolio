Installation d'un poste de développement

Présentation globale

Ce projet avait pour objectif d'installer un système d'exploitation Linux (Debian 11) et de configurer des outils de développement, dont un IDE (IntelliJ IDEA). Ce projet est divisée en deux parties :
- Installation d’un système d’exploitation (Debian) avec une interface graphique Plasma/KDE.
- Installation d’outils de développement (IDE, etc.).

Il s'agissait d'un projet individuel en autonomie. Pour l'installation du système d'exploitation (OS), les étapes sont les suivantes :
- L'installation sera effectuée sur une machine virtuelle (MV) stockée sur une station Linux.
- L'installation se fera via une interface graphique.

Les packages nécessaires seront installés à l'aide du gestionnaire de paquets "apt".

Ressources qui alimentent ce projet

Les ressources utilisées pour ce projet sont :
- R1.04 [Introduction aux systèmes d’exploitation et à leur fonctionnement]
Elle comprend mes connaissances sur le système Debian.

- R1.11 [Base de la communication]
Elle comprend nos méthodes de communication lors d'un contexte projet.

Mon rôle

Le travail étant individuel j'ai réalisé ce projet en autonomie et individuellement.

Mes compétences acquises

À la fin de ce projet, j'ai acquis plusieurs compétences :
- Maîtrise de l'installateur Debian.
- Installer un système d'exploitation Linux.
- Configurer un système d'exploitation pour l'adapter à des besoins spécifiques.
